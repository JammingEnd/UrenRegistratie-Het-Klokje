# Uren Registratie
## Project Klokje

### Werwijze
Wij maken gebruik van **GitFlow** om de repository schoon en gestructureerd te houden. Hieronder een korte uitleg van de verschillende branches en hun gebruik:

### Branches

- `main` - de branch met de laatste stabiele versie van de code.
- `develop` - de branch waar de laatste versie van de code staat, gebranch vanaf `main` .
- `feature/naam` - een branch voor elke nieuwe feature die ontwikkeld wordt, gebranch vanaf `develop`. (_De naam van de branch beschrijft de feature die ontwikkeld wordt._)
- `release/naam-v.1.0.0` - een branch voor het voorbereiden van een nieuwe release, gebranch vanaf `develop`. (_De naam van de branch beschrijft de versie van de release._)
- `hotfix` - een branch voor het snel oplossen van bugs, gebranch vanaf `main`.
